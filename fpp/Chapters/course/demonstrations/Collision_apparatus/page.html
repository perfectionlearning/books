  • Collision apparatus: Track with two carts

Demonstration 1: Elastic Collision of Objects—Equal Mass

In Demonstration 1, consider the momentum of two carts of equal mass before and after they collide head-on.

Question(s) for Students

When two carts of equal mass collide head-on and rebound on spring bumpers, how does the momentum of each cart before the collision compare with the momentum of each cart after the collision? How about the kinetic energy?

Identify the Variables

Mass, “m,” in kilograms and velocity, “v,” in meters per second.

Use the equations:

p = mv, where momentum is measured in kg·m/s

m1v1i + m2v2i = m1v1f + m2v2f (Conservation of Momentum)

, where kinetic energy is measured in kg·m2/s2, or joules

Steps in the Experiment

Create a Data Collection Chart

  1. Create a Data Collection Chart for three trials with the following column headings:
    1. Column 1: Cart #1, Cart #2
    2. Column 2: Mass
    3. Column 3: Velocity before collision
    4. Column 4: Momentum before collision (calculated)
    5. Column 5: Kinetic energy before collision (calculated)
    6. Column 6: Velocity after collision
    7. Column 7: Momentum after collision (calculated)
    8. Column 8: Kinetic energy after collision (calculated)

Set Up the Apparatus

  1. Place the track on a level surface.
  2. Level the track by placing a cart in the middle of the track and ensuring that it stays in place.
  3. If necessary, adjust the leveling screws on either end until the track is level.
  4. Attach one motion sensor to each end of the track (or a photogate to measure each cart’s velocity, if you are using photogates instead).
  5. Follow the manufacturer’s instructions to set up all hardware and software that accompanies the motion sensors or photogates.
  6. You will use the manufacturer’s data collection tool to collect and analyze your data.

Perform the Experiment

  1. Use the triple beam balance to obtain the mass of each cart and record this information. Use spring or rubber bumpers between the cars.
  2. Place a cart on each end of the track. Refer to the sensor and accompanying cart at one end of the track as #1 and the sensor and accompanying cart at the opposite end of the track as #2.
  3. One person should handle the carts, and the other the data collection tool.
  4. Agree on a countdown or similar signal so that cart launching and data collection can begin at the same time.
  5. Propel the two carts toward each other in a straight line.
  6. Stop collecting data when both carts have come to a complete stop.
  7. Be sure to save the data.
  8. Repeat steps 9−14 for two more trials.
  9. Save this setup for the next experiment.


Data Collection

Use the data collection tool to collect and analyze your data.

Question(s) for Students


Analyze Your Results


Calculate and Graph Your Results

  1. Use the software that accompanies the data collection tool to plot velocity vs. time graphs for each trial and for each cart before the collision. Save these graphs.
  2. Use the software that accompanies the data collection tool to plot velocity vs. time graphs for each trial and for each cart after the collision. Save these graphs.
  3. Use the equations: to calculate the momentum for each cart in each instance and record this value in the Data Collection Chart.


Compare and Contrast

  1. From the graphs (or from your photogate results), determine the velocity, momentum, and kinetic energy of each cart before and after the collision. Be careful of your minus signs!
  2. Compare the results of the three trials. Describe any differences and try to explain them.
  3. Review your observations. How did the behavior of the carts affect the results?
  4. What modification to the experimental procedure would you recommend to be sure the results were more consistent?
