Solving systems of linear equations (also called simultaneous equations):

A system of independent linear equations containing $n$ unknown variables can be solved when the

number of equations is $n$ or greater. One method of solving the system is:

1. Write the equations in the same form. Below we write both equations in the form $ax+by=c$.
$5x+(x-y)=3 \rightarrow 6x-y=3$
$2y=8+x$          $\rightarrow -x + 2y=8$

2. Multiply one equation by a constant so as to eliminate one variable when the equations are added. Below we multiply the lower equation by 6.

$6x-y=3$          $6x-y=3$
$-x+2y=8$  $\rightarrow -6x + 12y = 48$

3. Add the two equations and solve for the leftover variable.

+  6x−6x  −y+12y=3 =48    0x+11y =51 y= 5111

4. Substitute the value of the solved-for variable into one of the original equations to solve for the other variable.
$2y=8+x$
$x=2y-8$
$x=2(\cfrac{51}{11})-8$
$x = \cfrac{14}{11}$

Significant figures (significant digits)

The number of significant figures in a number indicates its precision.
Use the following rules to determine the number of significant figures:
1. All non-zero digits are significant.
2. Zeros that come between non-zero digits are significant.
3. Zeros that precede the first non-zero digit are not significant.
4. Zeros that come after the decimal point and after any non-zero digits are significant.

Numbers that end in zeros that precede the decimal point have an indeterminate number of significant figures. For instance, you cannot assume that 4500 has four significant figures, but you know that it has at least two.

More examples:

0.092 has two significant figures
0.0920 has three significant figures
0.090020 has five significant figures
10.08 has four significant figures
4000.000 has seven significant figures
4000 may have one, two, three or four significant figures
